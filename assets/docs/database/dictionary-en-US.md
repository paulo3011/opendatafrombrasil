
# #__Establishments__

| Field                          | Description                                                                                          |
| ---------------------------    | -------------------------------------------------------------------------------------------------- |
| BASIC CNPJ                     | BASE CNPJ REGISTRATION NUMBER (FIRST EIGHT DIGITS OF CNPJ).                                        |
| CNPJ ORDER                     | NUMBER OF THE ESTABLISHMENT (FROM THE NINTH TO THE TWELFTH DIGIT OF THE CNPJ).      |
| CNPJ DV                        | VERIFYING DIGIT OF THE CNPJ REGISTRATION NUMBER (LAST TWO DIGITS OF THE CNPJ). |
| HEADQUARTERS/BRANCH IDENTIFIER | IDENTIFIER CODE HEADQUARTERS/BRANCH:</br> 1 – HEADQUARTERS</br>2 – BRANCH |
 FANTASY NAME                    | THE FANTASY NAME |
| REGISTRATION STATUS            | CODE OF REGISTRATION STATUS:</br>01 – NULL</br>2 – ACTIVE</br>3 – SUSPENDED</br>4 – INAPTA</br>08 – DOWNLOADED |
| DATE REGISTRATION STATUS       | DATE OF REGISTRATION STATUS EVENT (YYYYMMDD) |
| REASON REGISTRATION STATUS | CODE OF REASON FOR REGISTRATION STATUS |
| NAME OF THE CITY ABROAD | NAME OF THE CITY ABROAD |
| COUNTRY | COUNTRY CODE |
| ACTIVITY START DATE | ACTIVITY START DATE (YYYYMMDD) |
| MAIN TAX CNAE | CODE OF THE MAIN ECONOMIC ACTIVITY OF THE ESTABLISHMENT |
| SECONDARY TAX CNAE | CODE OF THE SECONDARY ECONOMIC ACTIVITY(IES) OF THE ESTABLISHMENT |
| TYPE OF ADDRESS | DESCRIPTION OF THE TYPE OF ADDRESS |
| ADDRESS | NAME OF THE ADDRESS WHERE THE ESTABLISHMENT IS LOCATED. |
| NUMBER | NUMBER WHERE THE ESTABLISHMENT IS LOCATED. WHEN THE NUMBER IS NOT FILL IN, THERE WILL BE ‘Y/N’. |
| COMPLEMENT | COMPLEMENT TO THE ESTABLISHMENT'S LOCATION ADDRESS |
| DISTRICT | NEIGHBORHOOD WHERE THE ESTABLISHMENT IS LOCATED. |
| ZIP CODE | CODE OF POSTAL ADDRESS FOR THE ADDRESS IN WHICH THE ESTABLISHMENT IS LOCATED |
| UF | ACRONYM OF THE UNIT OF THE FEDERATION IN WHICH THE ESTABLISHMENT IS FOUND |
| CITY CODE | CODE OF THE MUNICIPALITY OF JURISDICTION WHERE THE ESTABLISHMENT IS FOUND (NOT THE IBGE CODE) |
| AREA CODE 1 | CONTAINS AREA CODE 1 |
| PHONE 1 | CONTAINS PHONE NUMBER 1 |
| AREA CODE 2 | CONTAINS AREA CODE 2 |
| PHONE 2 | CONTAINS PHONE NUMBER 2 |
| FAX DDD | CONTAINS FAX AREA CODE |
| FAX | CONTAINS FAX NUMBER |
| ELECTRONIC MAIL | CONTAINS TAXPAYER'S E-MAIL |
| SPECIAL SITUATION | SPECIAL COMPANY SITUATION |
| DATE OF SPECIAL SITUATION | DATE ON WHICH THE COMPANY ENTERED IN A SPECIAL SITUATION |
